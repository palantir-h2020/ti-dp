NetflowPreprocessingFunctions 

class NetflowPreprocessingFunctions extends Serializable

NetflowPreprocessingFunctions class. Contains all methods that will be used for preprocessing ingested raw netflow data to Kafka. Thr available operations of this class are the following: anonymize a netflow record, preprocess a netflow record. Anonymization is happening in a separate service, which exposes an API for sending IP addresses for anonymization and responds with the obfuscated IP address. Preprocessing operation creates some new features based on the existing ones of the netflow data.

  4.  def anonymizeRecord(anonymizationEndpoint: String, colsIdxToAnonymize: Array[Int], ipList: HashSet[String], isBenchmarkEnabled: Boolean): UserDefinedFunction

    Anonymize a netflow record.

    Anonymize a netflow record. Only specific (internal) IP addresses will be anonymized. These addresses are defined in ip-anonymize.txt file. The names of the columns of the netflow that must be anonymized are configured in netflow-preprocessing.properties file or through Dockerfile if the app is running containerized. For each of these columns, the app checks if it belongs in the list with the IP addresses that must be anonymized. If it belongs, an HTTP request is send to anonymization service. This service anonymizes the given IP and returns the obfuscated IP among other information and some stats about the query. The anonymized IP address replaces the original one and the netflow record is returned to the main Spark Streaming App in order to be sent back to Kafka after all Spark operations are completed. If app's benchmark mode is enabled both HTTP status code and response body of the anonymization service will be stored in benchmark.txt file.

    anonymizationEndpoint

    String Anonymization service endpoint, where the HTTP request will be sent. This must be in format http://IP:PORT/anonymize.

    colsIdxToAnonymize

    Array[Int] An array containing all indexes of columns that their values must be sent for anonymization.

    ipList

    HashSet[String] A set containing all IP addresses that must be anonymized. This is created after loading ip-anonymize.txt file which contains all subnets that their IPs must be anonymized. From these subnets, a HashSet of all IP addresses has been created and loaded.

    isBenchmarkEnabled

    Boolean Defines if the app is running in benchmark mode or not. If it is running in benchmark mode, anonymization service responds will be dumped in a file, named benchmark.txt, for analysis.

    returns

    String The input netflow record, but with the required IP addresses replaced with the obfuscated ones.

  23.  def preprocessRecord(): UserDefinedFunction

    Preprocess a netflow record.

    Preprocess a netflow record. Some new extra features will be created from the existing ones. The created new features are the following:
    tpkt: Total packets of the netflow. If the netflow is bidirectional this is the sum of ipkt and opkt. Otherwise, this is equal to ipkt
    tbyt: Total bytes of the netflow. If the netflow is bidirectional this is the sum of ibyt and obyt. Otherwise, this is equal to ibyt
    cp: Check if destination port is a port used from common services or not. Value is 1 if destination port is common, 0 otherwise
    prtcp: Checks if protocol is TCP. Value is 1 if it is, 0 otherwise.
    prudp: Checks if protocol is UDP. Value is 1 if it is, 0 otherwise.
    pricmp: Checks if protocol is ICMP. Value is 1 if it is, 0 otherwise.
    prigmp: Checks if protocol is IGMP. Value is 1 if it is, 0 otherwise.
    prother: Checks if protocol is not TCP, UDP, ICMP or IGMP. Value is 1 if it is none of them, 0 otherwise.
    flga: Checks if TCP flags of the netflow contain letter A. Value is 1 if it contains it or if TCP flag is equal to X, 0 otherwise
    flgs: Checks if TCP flags of the netflow contain letter S. Value is 1 if it contains it or if TCP flag is equal to X, 0 otherwise
    flgf: Checks if TCP flags of the netflow contain letter F. Value is 1 if it contains it or if TCP flag is equal to X, 0 otherwise
    flgr: Checks if TCP flags of the netflow contain letter R. Value is 1 if it contains it or if TCP flag is equal to X, 0 otherwise
    flgp: Checks if TCP flags of the netflow contain letter P. Value is 1 if it contains it or if TCP flag is equal to X, 0 otherwise
    flgu: Checks if TCP flags of the netflow contain letter U. Value is 1 if it contains it or if TCP flag is equal to X, 0 otherwise

    returns

    Netflow record, with the new created features added in the right of the old columns.
