ResponseBody 

class ResponseBody extends Serializable

ResponseBody class. A class describing the body of the response of the IP anonymization HTTP request. The body represents an object, that will be converted in JSON format. The following JSON keys will exist in this JSON: obfuscatedIp, originalIp, result, execution_time.
